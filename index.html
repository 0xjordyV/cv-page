2022 · Solana · P2E

Bitmon: Play-to-Earn Game

Pokémon-style P2E built with Kindynos, Vibing Studios, and Maki Studios. 10,000 Trainer PFPs gated the BitmonDAO. Players captured creatures in-game and minted them as real Solana NFTs. I built the Trainer Creator (pick attributes → mint on-chain), the $BIT token bridge (on-chain balance ↔ in-game currency), and the shared marketplace powered entirely by our own $BIT and $VIBE tokens.

P2E Game Trainer Creator $BIT Token Bridge NFT Marketplace $VIBE Token BitmonDAO 10k NFTs

2022–2023 · Solana · Anime IP

Vibe Knights: 8,888 Anime Elves

8,888 unique anime elf NFTs building a shared global IP, later rebranded to Zenjaku (3,050 after burns). I built the generation algorithm, main website, mint page with tiered rarity (Vibing Pass probability tables), an interactive "Art Map" roadmap, and a privacy-first holder world map, with locations randomized within a large radius so holders could find each other without being doxxed.

About

Mexican developer. I spent several years as CIO at Vibing Studios — a Web3 & Web2 game and art studio on Solana — shipping generative NFT pipelines, mint dapps, P2E mechanics, community tooling, and a viral $BONK activation.
